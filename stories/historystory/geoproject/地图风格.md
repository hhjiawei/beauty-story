这三个输出分别对应 **所有底图服务商**、**CartoDB 的子风格**、**OpenStreetMap 的子风格**。我按你的历史视频制作需求，把关键风格分类说明：

---

## 一、CartoDB 系列（你的主力底图）

这是最适合历史视频叠加的系列，**设计初衷就是"让上层数据突出"**：

| 名称 | 风格 | 适用场景 |
|------|------|---------|
| `Positron` | 浅灰白底 + **黑色现代地名** | 需要现代地名做地理参考 |
| `PositronNoLabels` | 浅灰白底 + **无任何文字** | ⭐ **历史视频首选**，干净画布 |
| `PositronOnlyLabels` | 透明底 + **只有文字** | 叠加在其他底图上做标注层 |
| `DarkMatter` | 深灰黑底 + 白色现代地名 | 暗黑风格、夜间战役 |
| `DarkMatterNoLabels` | 深灰黑底 + 无文字 | ⭐ **史诗暗黑风格首选** |
| `DarkMatterOnlyLabels` | 只有白色文字 | 标注层 |
| `Voyager` | 自然彩色（绿黄地形）+ 地名 | 需要地形色彩区分 |
| `VoyagerNoLabels` | 自然彩色 + 无文字 | 需要地形色彩但无现代干扰 |
| `VoyagerLabelsUnder` | 自然彩色，文字在图层下方 | 特殊叠加需求 |

**你的选择**：
- **浅色古风** → `PositronNoLabels`
- **暗黑史诗** → `DarkMatterNoLabels`

---

## 二、OpenStreetMap 系列

OSM 原生风格，**信息量大、颜色鲜艳**，不太适合直接做历史底图（现代感太强）：

| 名称 | 风格 |
|------|------|
| `Mapnik` | OSM 默认风格（彩色道路、建筑、绿地） |
| `HOT` | 人道主义风格（更简洁，淡色道路） |
| `DE` / `France` / `CH` / `CAT` / `BZH` | 德国/法国/瑞士/加泰罗尼亚/布列塔尼 本地化版本 |

**用途**：需要精确现代道路网时临时参考，**不建议直接做历史视频底图**。

---

## 三、其他一级服务商（值得关注的）

| 服务商 | 关键子风格 | 特点 |
|--------|-----------|------|
| **Esri** | `WorldImagery`（卫星影像）、`WorldTerrain`（地形晕渲） | 真实地形质感，适合做"山川形胜"分析 |
| **Stadia** | `StamenTerrain`（地形）、`StamenToner`（黑白）、`StamenWatercolor`（水彩） | ⭐ **Stamen 的合法继任者**，地形风格做历史地图非常好看 |
| **OpenTopoMap** | 默认 | 地形等高线晕渲，适合做关隘/山川战场 |
| **USGS** | `USTopo`（美国地形图） | 美式地形图风格 |
| **NASAGIBS** | `ModisTerraTrueColorCR`（卫星真彩色） | 真实地表覆盖 |
| **Gaode** | 默认 | 高德地图，国内速度快 |
| **Thunderforest** | `Landscape`（地形）、`Outdoors`（户外）、`Transport`（交通） | 英国地形风格，细节丰富 |
| **MapTiler** | 需要 API Key，多种现代风格 | 商业级，需注册 |
| **Strava** | `Heatmap`（运动热力） | 人类活动密度，可做"人口/迁徙"参考 |

## 四、特别推荐：Stadia 系列

你之前想用 `Stamen.TerrainBackground`，但 Stamen 已关闭公共访问。**Stadia 收购了 Stamen 的数据**，提供了几乎一致的风格：

```python
# Stadia 地形背景（无标签，类似旧版 Stamen Terrain）
ctx.providers.Stadia.StamenTerrainBackground

# Stadia 地形（带标签）
ctx.providers.Stadia.StamenTerrain

# Stadia 黑白线描（类似 Toner）
ctx.providers.Stadia.StamenToner

# Stadia 水彩（艺术风格）
ctx.providers.Stadia.StamenWatercolor
```

**注意**：Stadia 需要 **API Key**，免费额度足够个人使用。注册后在代码中：

```python
import os
os.environ["STADIA_API_KEY"] = "你的-api-key"

ctx.bounds2raster(
    bounds, output="base.tif", zoom=10,
    source=ctx.providers.Stadia.StamenTerrainBackground,
    ll=True
)
```

## 五、你的历史视频底图选择指南

| 视频风格 | 推荐 Provider | 理由 |
|---------|--------------|------|
| **正统历史纪录片**（浅色、严肃） | `CartoDB.PositronNoLabels` | 最干净，朱砂红/墨黑矢量层对比度极高 |
| **暗黑史诗/战争片头** | `CartoDB.DarkMatterNoLabels` | 深底+红金矢量，电影感强 |
| **山川形胜/关隘攻防** | `Stadia.StamenTerrainBackground` + Key | 真实地形晕渲，观众一眼看懂"为何在此设关" |
| **卫星视角/环境变迁** | `Esri.WorldImagery` | 真实地表，适合做"古今对比" |
| **人口迁徙/经济分布** | `CartoDB.VoyagerNoLabels` | 自然色彩区分平原/山地，叠加热力图清晰 |

## 六、快速测试代码（本地运行）

```python
import contextily as ctx
import matplotlib.pyplot as plt

bounds = [105.0, 30.0, 120.0, 42.0]

# 同时测试三种风格
styles = {
    "PositronNoLabels": ctx.providers.CartoDB.PositronNoLabels,
    "DarkMatterNoLabels": ctx.providers.CartoDB.DarkMatterNoLabels,
    "StamenTerrain": ctx.providers.Stadia.StamenTerrainBackground,  # 需 API Key
}

for name, source in styles.items():
    try:
        fig, ax = plt.subplots(figsize=(10, 8))
        ctx.add_basemap(ax, crs="EPSG:4326", source=source, zoom=8, ll=True)
        ax.set_title(name, fontsize=14)
        ax.axis('off')
        plt.savefig(f"test_{name}.png", dpi=150, bbox_inches='tight')
        print(f"✅ {name} 成功")
    except Exception as e:
        print(f"❌ {name} 失败: {str(e)[:50]}")
```

**总结**：你的 `Stamen` 报错是因为该服务商已关闭，**直接换 `CartoDB.PositronNoLabels` 或 `DarkMatterNoLabels`** 即可，无需 API Key、国内可访问、风格最适合历史叠加。
