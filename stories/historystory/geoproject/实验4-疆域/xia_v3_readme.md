# 夏桀时期数据包 v3.0 —— 占领 vs 附庸 修正版

## 核心修正

### 1. 占领 vs 附庸 的定义区分

| 类型 | 定义 | 数据表现 | 颜色表现 |
|------|------|---------|---------|
| **占领** | 武力征服，直接吞并领土 | owner=征服者, vassal_of=None | 征服者**正常色** |
| **附庸** | 主动归附/联盟，保持名义独立 | owner=附庸国, vassal_of=主国 | 主国**淡化色** |

### 2. 修正示例

**修正前（错误）**:
```json
// 商汤灭葛国
city: { "initial_owner": "商部落" }  // 初始就是商的，没有变化过程
event: { "from": "葛国", "to": "商部落", "vassal_of": "商部落" }  // 被征服后算附庸？
```

**修正后（正确）**:
```json
// city.json — 初始状态：葛国是独立方国
city: { "initial_owner": "葛国", "vassal_of": null }

// events.json — 商汤灭葛国：武力征服，直接占领
event: { "from": "葛国", "to": "商部落", "vassal_of": null, "type": "占领" }
// 结果：葛国区域显示商部落正常色（深蓝），不是附庸色
```

### 5. 颜色渲染规则

```python
def get_color(owner, vassal_of, color_map):
    if vassal_of is not None and vassal_of in color_map:
        # 附庸：使用主国的淡化色（如夏朝附庸 = 浅红棕 #C5927F）
        return color_map[vassal_of]["vassal_fill"]
    else:
        # 占领/独立：使用正常色（如商部落 = 深蓝 #1B4F72）
        return color_map[owner]["fill"]
```

## 一县多城冲突解释

### 什么是"一县多城"？

**现代一个县级面内，可能包含多个古代城池。**

**示例**：山东省枣庄市滕州市（现代县级面）
- 古代城池A：有施氏（lat=35.0884, lon=117.1597）
- 古代城池B：薛国（lat=35.0700, lon=117.1800）
- 古代城池C：滕国（lat=35.0880, lon=117.1700）

这三个城池的坐标都在滕州市范围内，空间关联后都会映射到同一个县级面（county_idx = 滕州市的索引）。

### 为什么会冲突？

**不同城池可能归属不同国家。**

**场景**：
- 初始状态：有施氏（独立）、薛国（独立）、滕国（独立）→ 三个不同国家
- 但空间关联后，这三个城池都落在同一个县级面内
- 在 dissolve（合并）时，这个县级面只能有一个归属颜色

### 如何解决？

**优先级规则**：

```python
PRIORITY = {"都城": 4, "郡治": 3, "县治": 2, "关隘": 1, "要塞": 0}

def resolve_county_owner(city_ids, cities_state):
    # 1. 收集该县内所有城池
    cities_in_county = [cities_state[cid] for cid in city_ids]

    # 2. 按优先级排序（都城 > 郡治 > 县治 > 关隘 > 要塞）
    # 3. 同优先级按最新事件时间排序（最新优先）
    cities_in_county.sort(
        key=lambda c: (PRIORITY[c["type"]], c["last_event_time"]),
        reverse=True
    )

    # 4. 优先级最高的城池决定该县归属
    return cities_in_county[0]["owner"]
```

**示例**：
- 滕州市内有：有施氏（都城，priority=4）、薛国（都城，priority=4）、滕国（都城，priority=4）
- 三者优先级相同，取最新事件改变的城池
- 如果薛国最近被商汤征服（last_event_time 最新），则滕州市归属薛国（即商部落）

### 视觉表现

- 县级面显示为一个颜色（由优先级决定）
- 城池点（Circle 层）显示各自的真实归属颜色
- 观众通过城池点颜色可以感知到"该县有争议"

# 把 city_xia.json 的城池点映射到 geoBoundaries 县级面是啥意思 会产生什么 有啥用
**一句话解释**：就是判断"这个古代城池，落在今天哪个县的范围内"。

---

## 为什么要做这个映射？

你 `city_xia.json` 里有 40 个城池，每个只有**一个点**（经纬度坐标）。但你要在地图上显示的是**一片区域**（疆域面），不是一个点。

**问题**：这 40 个点各自管多大地方？怎么从"点"变成"面"？

**解决**：用现代的县级边界来"借地"。每个城池点落在哪个现代县里，这个县就归这个城池的势力管。

---

## 具体怎么做？（空间关联）

```python
# 你的 40 个城池点 → 落在哪个现代县级面里？
GeoPandas.sjoin(城池点, 县级面, predicate='within')
```

**直观例子**：

```
现代县级面（ geoBoundaries 数据）
┌─────────────────────────────────────┐
│  河南省洛阳市偃师区                    │
│                                     │
│    ● 斟鄩（lat=34.6997, lon=112.6917）│  ← 这个点落在偃师区里面
│                                     │
└─────────────────────────────────────┘

结果：城池"斟鄩" → 对应县级面索引 1234（偃师区）
```

**每个城池都会得到一个 `county_idx`** —— 表示它落在第几个县级面里。

---

## 会产生什么结果？

### 结果 1：每个城池多了 `county_idx`

```json
{
  "id": "xia_zhenxun",
  "ancient_name": "斟鄩",
  "lat": 34.6997,
  "lon": 112.6917,
  "county_idx": 1523,   // ← 新增！落在第1523个县级面里
  "initial_owner": "夏朝"
}
```

### 结果 2：`county_city_map.json`（反向映射）

```json
{
  "1523": ["xia_zhenxun"],           // 第1523个县 → 只有斟鄩
  "2841": ["youshi", "xueguo"],      // 第2841个县 → 有施氏 + 薛国（都在滕州市）
  "3105": ["shang_bo"],               // 第3105个县 → 只有亳都
  ...
}
```

**关键**：一个县可能包含**多个城池**（比如滕州市里有有施氏和薛国），这就是之前说的"一县多城"冲突。

---

## 有什么用？

### 用途 1：决定每个县归谁管

你有 2800+ 个县级面，但只关心有城池的那些。通过映射：

- 第 1523 个县里有夏朝都城斟鄩 → 这个县归 **夏朝**
- 第 2841 个县里有有施氏 + 薛国 → 按优先级决定归谁（都城优先级相同，取最新事件）

### 用途 2：生成疆域面（核心）

```python
# 所有归夏朝的县 → 合并成一个面
夏朝疆域 = dissolve(所有 owner="夏朝" 的县级面)

# 所有归商部落的县 → 合并成一个面  
商部落疆域 = dissolve(所有 owner="商部落" 的县级面)
```

**效果**：从 2800 个碎片县，合并成 7-10 个国家的疆域面，每个国家一块连续的区域。

### 用途 3：事件驱动变色

当 `events_xia.json` 里发生"商汤灭葛国"时：
1. 找到葛国对应的 `county_idx`（比如第 1802 个县）
2. 把这个县的 `owner` 从"葛国"改成"商部落"
3. 重新合并所有县 → 商部落的疆域面就多了一块
4. 输出新的 GeoJSON → 地图上商部落的蓝色区域变大了

## 总结：三步走

| 步骤 | 输入 | 操作 | 输出 |
|------|------|------|------|
| **1** | 40 个城池点 + 2800 个县级面 | 判断每个点落在哪个面里 | 每个城池的 `county_idx` |
| **2** | `county_idx` 反向映射 | 建立"县 → 城池列表" | `county_city_map.json` |
| **3** | 事件驱动归属变化 | 按优先级决定每个县归谁 | 时间断面 GeoJSON（疆域面） |
