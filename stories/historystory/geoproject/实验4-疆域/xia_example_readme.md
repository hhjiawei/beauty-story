## 数据使用方式

### 1. 空间关联（阶段1）

```python
import geopandas as gpd
from shapely.geometry import Point
import json

# 读取 city.json
with open('city_xia.json', 'r', encoding='utf-8') as f:
    cities = json.load(f)

# 转为 GeoDataFrame
cities_gdf = gpd.GeoDataFrame(
    cities['cities'],
    geometry=[Point(c['lon'], c['lat']) for c in cities['cities']],
    crs='EPSG:4326'
)

# 读取 geoBoundaries 县级面
counties = gpd.read_file('china_adm3.geojson')

# 空间关联
joined = gpd.sjoin(cities_gdf, counties, predicate='within', how='left')

# 结果中 index_right 即为所属县级面索引
for idx, row in joined.iterrows():
    print(f"{row['ancient_name']:8s} → 县级面索引: {row['index_right']}")
```

### 2. 时间断面生成（阶段2）

运行 `02_generate_slices.py`，输入：
- `city_xia.json`（城池坐标 + initial_owner）
- `events_xia.json`（事件序列）
- `color_map_xia.json`（配色）
- `china_adm3.geojson`（县级面数据）

输出：`time_slice_前1652年.geojson` ~ `time_slice_前1600年.geojson`（共 19+1 个文件）

### 3. 前端渲染（阶段4）

MapLibre 加载时间断面 GeoJSON，与行军路线联动：

```javascript
map.addSource('territory', {
  type: 'geojson',
  data: '/data/time_slice_前1652年.geojson'
});

map.addLayer({
  id: 'territory-fill',
  type: 'fill',
  source: 'territory',
  paint: {
    'fill-color': ['get', 'color'],
    'fill-opacity': ['get', 'fill_opacity']
  }
});
```

## 注意事项

1. **坐标精度**：部分方国（如九夷各部、防风氏、三苗）的地理位置为学术推测，confidence 标记为 `low`，建议渲染时降低透明度或使用虚线边框。

2. **一县多城冲突**：山东滕州市同时包含「有施氏」和「薛国」，空间关联后会落入同一县级面。按优先级规则（都城 > 郡治 > 县治 > 关隘 > 要塞），如两者类型相同，以最新事件为准。

3. **境外城池**：岷山氏（四川）、三苗（湖南/湖北）位于现代中国境内，但远离夏朝核心战区。空间关联会成功，但渲染时可能因距离过远导致视觉分散。

4. **时间跨度**：夏桀在位约 52 年（前1652-前1600），19 个事件覆盖了整个末期。如需更细粒度（如每年一个断面），可在事件间插入无变化年份。
