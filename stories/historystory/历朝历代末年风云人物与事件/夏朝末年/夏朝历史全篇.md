完整世系与故事线
禹（姒姓，名文命，父鲧，治水13年，三过家门不入，定九州，铸九鼎）→ 传位给益，但启夺位（一说启杀益，一说益让位）→ 启伐有扈氏（甘之战）→ 启在位时武观之乱（幼子叛乱）→ 启死，长子太康立 → 太康失国，后羿（有穷氏）夺权 → 后羿立太康弟仲康为傀儡 → 仲康死，子相立 → 后羿被家臣寒浞所杀，寒浞夺权 → 寒浞派子浇杀相 → 相妻后缗逃归有仍氏，生遗腹子少康 → 少康中兴，灭寒浞复国 → 少康子杼（作甲）东征东夷 → 杼子槐，九夷来朝 → 槐子芒，沉祭于河 → 芒子泄，赐封九夷 → 泄子不降，在位59年，传弟扃 → 扃子廑（胤甲），迁都西河 → 廑死，不降子孔甲立（孔甲乱夏，四世而陨）→ 孔甲子皋 → 皋子发 → 发子桀（履癸），杀贤臣关龙逄，囚商汤于夏台后释之 → 商汤伐桀，鸣条之战，桀败亡流放南巢，夏亡（约公元前1600年）。
