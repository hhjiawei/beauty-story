# 《帝国黄昏》历史视频号 — 完整策划方案

> **版本**：v1.0  
> **策划定位**：音频叙事 + 历史图片视频的短视频号  
> **核心模式**：每集一个核心场景 × 一个核心人物  
> **时间跨度**：黄帝时期 → 解放战争结束（约公元前26世纪 ~ 1949年）  
> **总集数规划**：约 80-100 集（每季 20 集，共 4-5 季）

---

## 第一部分：项目定位与产品策略（PRD核心）

### 1.1 项目一句话定位

> **"用说书人的口吻，讲透每个朝代最后一根稻草的故事——让历史像 bedtime story 一样好听。"**

### 1.2 目标受众画像

| 维度 | 描述 |
|------|------|
| **核心人群** | 25-45岁，对历史感兴趣但研究不深的上班族 |
| **场景** | 睡前30分钟、通勤路上、家务背景音乐 |
| **心理需求** | ① 听故事放松 ② 获得"历史谈资"社交货币 ③ 感受历史人物命运共鸣 |
| **付费意愿** | 中（愿意为优质内容付费/打赏） |
| **互动偏好** | 喜欢评论"原来如此""涨知识了"，愿分享至朋友圈 |

### 1.3 内容差异化策略

| 维度 | 我们的做法 | 传统历史号的做法 |
|------|-----------|----------------|
| **叙事单位** | 单人物 + 单场景 | 全景扫描式通史 |
| **语调** | 说书人/ bedside storyteller | 教授讲课/纪录片旁白 |
| **情感线** | 强（君臣、男女、父子羁绊） | 弱（事件为主） |
| **趣味元素** | 历史大瓜 + 反常识钩子 + 现代类比 | 年代罗列 + 政策分析 |
| **节奏** | 5分钟一集，有呼吸感 | 15-20分钟高密度 |
| **连续性** | 人物链串联，集集相扣 | 独立成章 |

### 1.4 发布平台与形式

| 平台 | 形式 | 频次 |
|------|------|------|
| **抖音/快手** | 音频+历史图片轮播视频 | 每周3-4更 |
| **视频号** | 同上 + 公众号文章沉淀 | 同步 |
| **小宇宙/喜马拉雅** | 纯音频播客版 | 同步 |
| **小红书** | 图文笔记（人物关系图） | 每周1-2更 |

## 第二部分：完整人物链设计（80集总纲）

> **设计原则**：
> 1. 严格按时间线推进
> 2. 每集末尾引入下一集人物（历史羁绊/因果关系）
> 3. 优先选择有"人性复杂度"的人物（非脸谱化）
> 4. 男女搭配、君臣搭配、文武搭配，保持节奏变化

## 第三部分：单集叙事模板（播客级脚本结构）

### 3.1 单集基本信息

| 项目 | 设定 |
|------|------|
| **目标时长** | 4-6 分钟（视频号最佳完播时长） |
| **字数** | 800-1200 字 |
| **语速** | 每分钟约 200 字（舒缓说书节奏） |
| **BGM** | 古琴/琵琶轻音乐为主，紧张情节加入鼓点 |
| **图片** | 5-8 张：①人物画像 ②场景还原图 ③文物/遗址照片 |

### 3.2 脚本结构（五段式）

#### 第一段：Hook 钩子（前20秒 ≈ 60字）

**功能**：抓住注意力，制造"好奇心缺口"

**六种钩子模板**：

| 类型 | 示例 |
|------|------|
| **反常识型** | "你知道吗？中国历史上被黑得最惨的女人，可能不是妲己，而是褒姒。" |
| **悬念型** | "公元前771年的那个夜晚，骊山脚下燃起了狼烟。但这一次，真的没有人来了。" |
| **现代类比型** | "如果把崇祯皇帝放在今天，他大概是一个每天工作16小时、却依然被董事会炒掉的CEO。" |
| **情感共鸣型** | "你有没有试过，明明已经拼尽全力，结局却还是一败涂地？1627年的朱由检，就是这样。" |
| **数字冲击型** | "1644年，北京城破的那一天，33岁的皇帝在煤山上吊自杀。他身边，只剩下一个太监。" |
| **场景代入型** | "想象一下，你是一国之君，拥有万里江山。但此刻，你正躲在一口枯井里，等着敌人来决定你的命运。" |

#### 第二段：人物登场（60秒 ≈ 200字）

**功能**：建立人物画像，让观众"看见"这个人

**写作要点**：
- 不只说"他是谁"，更要说"他像什么"
- 用一个具体细节让人物立体
- 交代人物在时代中的位置

**示例**（以崇祯为例）：
> "朱由检，明朝最后一位皇帝。他十六岁登基，二十岁铲除魏忠贤，三十三岁自缢身亡。史书上说他'性多疑而刚愎'，但如果翻译成现代话，大概就是一个有洁癖的完美主义者——每天批阅奏折到深夜，衣服打补丁，连后宫都很少去。问题是，越努力，越不幸。"

#### 第三段：核心场景（2-3分钟 ≈ 500字）

**功能**：讲述本期最核心的一个场景/事件，要有画面感

**写作要点**：
- 只讲一个场景，不要贪多
- 要有时间、地点、动作、对话
- 在场景中加入"小人物视角"
- 制造紧张感和节奏变化

**示例**（崇祯煤山自缢）：
> "1644年三月十九日凌晨，天还没亮。崇祯皇帝穿着蓝色大袍，赤着脚，在太监王承恩的搀扶下，走出了紫禁城。他们穿过空荡荡的街道，一路向北，来到了煤山——就是今天的景山。山上有棵老槐树。崇祯从袖中取出三尺白绫，看了一眼南方的京城。城外，李自成的军队已经入城，火光映红了半边天。他对王承恩说：'朕凉德藐躬，上干天咎，致贼直逼京师，皆诸臣误朕。朕死，无面目祖宗。'说完，他把头发披散下来遮住脸，意思是：我没脸见祖宗。然后，自缢身亡。他死时，身边只有王承恩一个人。三天后，人们才在树上发现了他的遗体。"

#### 第四段：深度解读（60秒 ≈ 200字）

**功能**：提供历史洞察，让观众"有所思"

**写作要点**：
- 提出一个反常识观点或深度解读
- 联系现代，让观众产生共鸣
- 不煽情，不说教，像一个有见识的朋友在聊天

**示例**：
> "崇祯的悲剧在于，他是一个好人，却不是一个好皇帝。他勤政、节俭、不近女色——但唯独缺少一样：识人之明和容人之量。十七年换了五十个内阁首辅，平均每个任期四个月。就像一个焦虑的CEO，不断换CFO，却从不想是不是自己的战略出了问题。所以明朝的灭亡，不是因为他懒，而是因为他太急。"

#### 第五段：过渡钩子（结尾30秒 ≈ 100字）

**功能**：自然引出下一集人物，形成追更动力

**模板**：
> "崇祯死了，但故事还没完。那个打开山海关、放清军入关的人——吴三桂，此刻正在赶来的路上。而他之所以做出这个决定，据说是因为一个女人。下一集，《冲冠一怒为红颜：吴三桂的双面人生》。"

### 3.3 语气与节奏指南

| 元素 | 规范 |
|------|------|
| **人称** | 以"我"（说书人）的口吻叙述，偶尔用"你"拉近听众 |
| **句式** | 长短结合，紧张情节用短句，抒情段落用长句 |
| **幽默** | 每集至少2处轻松幽默（现代类比、自嘲式点评） |
| **停顿** | 关键信息后用省略号标注停顿位置（如"然后，自缢身亡。……三天后，人们才发现了他的遗体。"） |
| **呼吸感** | 每90秒设置一个"情绪转换点"（紧张→放松→思考→紧张） |
| **禁忌** | ①不用文言文引用 ②不堆砌年代数字 ③不做价值判断（如"他是好人/坏人"） |

## 第五部分：幽默叙事技巧库

### 5.1 十种"历史大瓜"挖掘术

| 技巧 | 说明 | 示例 |
|------|------|------|
| **人设反差法** | 展示人物不为人知的一面 | 秦始皇其实是一个工作狂，每天批阅120斤竹简 |
| **现代类比法** | 用现代概念解释古代行为 | 隋炀帝修大运河，就像一个CEO同时启动十个项目 |
| **吐槽法** | 以现代人的视角吐槽古人 | 王莽的改革，简直就是古代版"计划经济乌托邦" |
| **细节放大法** | 聚焦一个荒诞/有趣的细节 | 崇祯皇帝的衣服打补丁，但补丁比原来的衣服还贵 |
| **假设法** | "如果……会怎样" | 如果柴荣多活十年，可能就没有赵匡胤什么事了 |
| **八卦法** | 挖掘人物的感情纠葛 | 萧皇后被六个男人争抢，但她真正爱的可能是杨广 |
| **数据冲击法** | 用具体数字制造反差 | 和珅的家产，相当于清朝15年的财政收入 |
| **对比法** | 同一事件的不同视角 | 李自成进北京时，以为自己是刘邦；离开时，发现自己只是黄巢 |
| **冷知识法** | 挖掘鲜为人知的史实 | 烽火戏诸侯可能是后世编的，西周灭亡的真正原因是继承人争夺 |
| **预言法** | 引用当时人的预言或后世评价 | 太史伯阳父早在十年前就预言了西周灭亡 |

### 5.2 节奏把控：呼吸感设计

| 时间节点 | 情绪曲线 | 操作方式 |
|----------|---------|---------|
| 0-20秒 | 好奇（Hook） | 抛出悬念 |
| 20-80秒 | 轻松（人物登场） | 幽默介绍，建立人物画像 |
| 80-200秒 | 紧张（核心场景） | 快节奏叙事，短句为主 |
| 200-230秒 | 舒缓（解读） | 放慢语速，引发思考 |
| 230-260秒 | 期待（过渡） | 预告下一集，制造追更欲望 |

### 5.3 避免的雷区

| 雷区 | 正确做法 |
|------|---------|
| ❌ 把历史讲成道德课 | ✅ 展现人物的复杂性，不说"好人/坏人" |
| ❌ 过度使用网络梗 | ✅ 适度类比，保持历史庄重感 |
| ❌ 编造史实 | ✅ 明确区分"正史记载""野史传说""推测" |
| ❌ 贬低女性角色 | ✅ 重新审视"红颜祸水"论，展现女性能动性 |
| ❌ 汉族中心主义 | ✅ 多视角呈现（少数民族、底层民众） |
| ❌ 过度简化因果 | ✅ 展现历史的多元因果（政治+经济+气候+个人） |
