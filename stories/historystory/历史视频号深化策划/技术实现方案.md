# 二、脚本获取资料
搞一个定型的资料清单对象，写作前需要准备相关资料，可以根据清单来搜集相关资料
1）本地向量库 
2）网络搜索  （网上资料网站 ）


# 三、生成脚本
1）脚本写作资料 （人 朝代 人际关系 事件 等）
2）写作风格评估 根据写作资料进行评估，使用哪种写作风格 这种风格的写作技巧或者技术等相关信息准备
3）结合资料 在该风格下 如何写 梳理大纲
4）开始写作脚本
5）评审 -> 复写
6）检查语法、错别字、历史事件是否正确 
7）初稿脚本选择背景音乐、背景视频
8）初稿通过tts转为语音
9）文字 语音、背景音乐、背景视频素材整合
