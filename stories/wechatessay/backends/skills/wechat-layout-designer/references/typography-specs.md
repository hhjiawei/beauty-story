# 排版技术规范

微信公众号编辑器内的具体参数标准。所有数值基于移动端阅读体验优化。

## 字号体系

| 层级 | 字号 | 行高 | 字重 | 使用场景 |
|------|------|------|------|---------|
| H1 | 18px | 1.8 | bold | 文章主标题 |
| H2 | 16px | 1.8 | bold | 章节标题、核心观点 |
| H3 | 15px | 1.8 | bold | 小标题、步骤标题 |
| Body | 14px | 1.8 | normal | 正文（微信默认） |
| Caption | 12px | 1.6 | normal | 注释、出处、灰色辅助 |
| Hook | 15-16px | 2.0 | bold | 引言钩子（需突出时） |
| Quote | 15px | 1.8 | normal | 引用/金句框内文字 |

字号变化规则：相邻层级至少差2px才有感知。正文中出现大号字（Hook级）视为"强调突变"，每篇文章不超过3处。

## 色彩系统

### 主色调选择（根据内容情绪）

| 情绪/调性 | 推荐主色 | 应用场景 |
|----------|---------|---------|
| 专业/信任 | #2E5BFF（蓝） | 知识干货、数据分析 |
| 警示/紧迫 | #FF4D4F（红） | 避坑、错误纠正 |
| 温暖/共鸣 | #FF7A45（橙） | 故事、情感 |
| 成长/正向 | #52C41A（绿） | 成功案例、方法论 |
| 高端/克制 | #262626（近黑） | 观点评论、深度文 |
| 辅助/柔和 | #8C8C8C（灰） | 注释、次要信息 |

### 使用规范
- 正文颜色：#262626 或 #595959（避免纯黑#000）
- 高亮颜色：主色用于加粗、标题、边框，每篇不超过2个主色
- 背景色块：主色10%透明度（如 #2E5BFF1A）
- 注释/次要：#8C8C8C
- 禁用颜色：#BF0000（刺眼）、#00FF00（廉价）

## 留白规范

### 段间距
- 段与段之间：至少1.5倍行高（约21px）
- 章节之间：2-3倍行高（约40-60px），可插入分隔线
- 金句框上下：各留2倍行高空白

### 行内呼吸
- 正文中每段控制在2-4行（约60-120字）
- 超过4行必须拆段或在段中插入视觉元素
- 列表项之间间距1.2倍行高

### 边距
- 正文左右边距：使用微信默认（约15px）
- 引用/卡片左右内边距：16px
- 金句框左右内边距：20px（比正文缩进，制造聚焦）

## 高亮锚点设计

### L1 金句框（最强）
```
视觉规格：
- 独立成块，前后各空2行
- 背景色：主色10%透明度
- 左边线：4px 主色实线
- 字号：15-16px（比正文大1-2级）
- 文字：可加引号，可居中或左对齐
- 圆角：4px
```

### L2 行内高亮
```
视觉规格：
- 加粗 + 主色文字
- 或加粗 + 主色底色高亮（文字不变色，背景主色15%透明度）
- 不改变字号
- 每段不超过1处
```

### L3 常规强调
```
- 纯加粗（文字颜色不变）
- 下划线（少用，易显脏）
```

## 分隔元素库

| 元素 | 符号/样式 | 使用场景 |
|------|----------|---------|
| 呼吸空行 | 纯空白（1.5-2行高） | 段间自然停顿 |
| 轻分隔线 | `— — —` 居中灰色 | 章节间过渡 |
| 时间跳转 | `· · ·` 或 `— 三年后 —` | 故事时间线跳转 |
| 转折点 | `▶` 或 `→` + 灰色 | 逻辑推进提示 |
| 装饰分隔 | 小图标/星号居中 | 情绪段落间 |

## 移动端适配原则

- 一行不超过25个汉字（14px下约350px宽度）
- 任何元素最小点击/感知区域不低于44px
- 卡片式内容左右边距不小于12px（避免贴边）
- 深色背景文字必须用纯白#FFFFFF，对比度>4.5:1
- 避免使用纯蓝色（#0000FF）作为链接色，与微信默认链接混淆
