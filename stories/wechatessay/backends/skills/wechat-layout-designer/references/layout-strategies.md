# 四型内容排版策略（Step 2）

根据Step 1诊断结果，选择对应策略组合应用。

## 叙事故事型 —— 情绪流排版

核心目标：让读者沉浸，随情节起伏。

### 结构模板
```
【钩子区】1-2句强冲突引言 → 大号字/异色/居中
【背景线】交代时间地点 → 小号灰色/营造氛围
【起伏段】情节推进 → 常规正文，长短句交替
  ↓ 低谷处插入「呼吸空行」或「时间跳转符」···
【高潮区】情感爆发点 → 金句框/居中大/L1高亮
【留白区】情绪沉淀 → 大面积空行 + 极简短句
【收尾段】余韵或反转 → 斜体/灰色/小字号
```

### 关键技巧
- 用字号变化制造"远近感"：回忆/背景缩小，冲突/高潮放大
- 对话单独成段，加引号框或不同底色
- 时间跳转用「···」或「— 三年后 —」作为视觉分隔
- 结尾必须有"余韵设计"：大面积留白+一句低饱和度的话

## 知识干货型 —— 认知减负排版

核心目标：降低理解成本，制造"我学到了"的爽感。

### 结构模板
```
【痛点钩子】读者现状痛点 → 加粗反问句
【承诺区】读完能获得的收益 → 编号/绿色/清单式
【正文层】方法论展开
  ├─ 每个步骤独立成块
  ├─ 步骤编号用圆形色块（1/2/3）
  ├─ 核心概念用「概念卡」高亮
  └─ 难点用「注意框」警示
【案例区】一个具体案例验证 → 浅灰底色/左边线
【总结卡】3-5条核心 takeaway → 列表/勾选框样式
【CTA区】互动提问或行动号召 → 底部框/异色
```

### 关键技巧
- 每300字必须有一个"认知锚点"（加粗句/概念卡/编号）
- 方法步骤用视觉编号（圆形色块 > 数字括号）
- 抽象概念一律配「举例框」：灰色底色，"比如：xxx"
- 结尾必须有takeaway清单，制造收获感

---

## 观点评论型 —— 论证力度排版

核心目标：让立场鲜明，论证有层次，读者愿意站队。

### 结构模板
```
【立场锚】核心观点 → 独立成段/大号/加粗/居中或左对齐
【论据链】逐层论证
  ├─ 每个论点独立段落
  ├─ 论点句加粗前置（先亮观点，再给解释）
  └─ 反对意见用「引用框」+ 反驳
【金句锤】最强论证 → L1金句框/异色底/大字
【升华段】价值观拔高 → 留白+递进排比
【站队区】引导表态 → 底部互动提问
```

### 关键技巧
- 核心观点必须在前3段出现，不绕弯
- 采用"论点加粗前置"格式：`**核心判断。** 解释说明...`
- 反对声音用引用样式包裹，再跟反驳（制造辩论张力）
- 排比句每句独立成行，左对齐，制造递进感
- 结尾制造"站队"冲动：让认同的读者想转发表态

## 清单攻略型 —— 扫描式排版

核心目标：适配"先收藏后看"习惯，让每条都清晰可扫。

### 结构模板
```
【价值锚】总量预告 → "X个技巧/工具/方法" → 异色大字
【适用区】这篇适合谁 → 标签化/人群图标
【清单体】逐项展开
  ├─ 每条独立卡片/色块
  ├─ 编号 + 标题 + 简述 + （可选）展开详情
  └─ 重要条目加「推荐」角标
【对比区】同类对比表 → 表格/左右分栏
【总结条】一句话总结 → 底部固定bar样式
```

### 关键技巧
- 每条必须有独立视觉边界（卡片/分割线/编号色块）
- 编号用渐变色块或emoji替代纯数字
- 每条控制在80-100字以内，超出则折叠为"展开"
- 关键条目标红/加推荐角标
- 必须有"收藏引导"设计：顶部或底部提示"建议收藏"

## 通用急救包：跨类型通用技巧

当诊断发现以下问题时，针对性插入：

| 诊断问题 | 排版解法 |
|---------|---------|
| 连续3段无视觉变化 | 插入「呼吸空行」或「分隔线」 |
| 120字以上大段 | 拆段 + 关键词加粗 + 概念卡 |
| 术语/信息过载 | 抽离为「术语卡」或「数据框」 |
| 情绪高潮处 | L1金句框：居中大/异色底/加引号 |
| 逻辑转折处 | 插入「转折符」→ 或 ▶ 提示 |
| 结尾无力 | 极简留白 + 一句低饱和金句 + CTA框 |
| 开头平淡 | 前置「钩子区」：2句冲突/反直觉表述 |

## 元素优先级（从高到低）

一篇完整排版中，以下元素按优先级选择性使用：

1. **L1金句框**（必须有，1-3处）
2. **呼吸空行/分隔**（必须有，每隔2-3段）
3. **编号/步骤视觉**（干货/清单必须有）
4. **概念卡/数据框**（信息密处必须有）
5. **CTA框**（结尾必须有）
6. **引用/举例框**（知识型适用）
7. **表情/图标点缀**（清单型适用，不超过5个）
