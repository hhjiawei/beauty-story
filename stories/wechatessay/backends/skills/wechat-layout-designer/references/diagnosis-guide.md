# 阅读体验扫描指南（Step 1）

对原文进行结构化诊断，输出四维度分析报告。

## 1. 内容类型判定

通读全文，根据主导特征归入以下四类之一：

| 类型 | 判定标准 | 关键词特征 |
|------|---------|-----------|
| 叙事故事型 | 有时间线/人物/情节/情感起伏 | 那年、后来、突然、没想到、永远记得 |
| 知识干货型 | 有方法论/步骤/数据/结论 | 第一步、核心逻辑、数据表明、关键在于 |
| 观点评论型 | 有立场/论证/批判/价值判断 | 我认为、本质上、问题在于、不可否认 |
| 清单攻略型 | 条目化/枚举/实用导向 | N个技巧、必备清单、避坑指南、合集 |

混合类型处理：找出占比超过60%的主导类型，以主导类型为骨架，其他类型段落按对应策略穿插处理。

## 2. 情绪/认知曲线标注

将全文按自然段落为单位，标注每段的读者状态：

### 高潮点（★）—— 必须强化
- 核心结论或顿悟时刻
- 情感共鸣最强处（感动/愤怒/惊喜）
- 最易引发转发的金句所在
- 数据最有冲击力的地方

### 低谷点（▼）—— 必须打断
- 信息密度过高、术语堆砌
- 叙事平淡、缺乏情感波动
- 逻辑转折生硬、理解成本高
- 连续超过3段无视觉变化

标注格式示例：
```
第1段：引言，情绪平稳 →  baseline
第2段：冲突出现，紧张 ↑ →  ramp
第3段：金句爆发，共鸣 ★ →  PEAK（L1高亮锚点）
第4段：背景解释，略平淡 ▼ →  valley（需插入呼吸符）
第5段：方法论，信息密 ▼ →  valley（需拆分+图解）
第6段：案例验证，情绪回 ↑ →  ramp
第7段：总结号召，行动力强 ★ →  PEAK（CTA强化）
```

## 3. 核心记忆点提取

从原文中精选1-3句话作为L1级高亮锚点。选择标准：

1. **截图价值**：读者看到会想截图发朋友圈
2. **转发价值**：脱离上下文仍有传播力
3. **情绪峰值**：承载最强情感或认知冲击
4. **字数控制**：单句不超过40字，便于一眼读完

提取后标注等级：
- L1（最高）：独立成块、视觉最重、必须一眼看到
- L2（次高）：加粗或变色、在正文内突出
- L3（辅助）：常规强调，不额外处理

## 4. 阅读黑洞段分析

逐段扫描，找出问题段落。黑洞段判定标准（满足任一）：

- 连续文字超过120字无断行
- 一个段落内包含3个以上概念/知识点
- 超过5行无视觉元素（无加粗、无引用、无列表、无分隔）
- 术语密度过高（每30字出现1个以上专业术语）

黑洞段修复策略（匹配使用）：

| 问题类型 | 修复方式 |
|---------|---------|
| 纯文字过密 | 拆段 + 插入呼吸空行 + 关键词加粗 |
| 多概念并列 | 改为列表/卡片式排版 |
| 术语堆砌 | 术语变色 + 插入通俗解释框 |
| 逻辑复杂 | 插入流程图/步骤条/分层缩进 |
| 数据密集 | 抽离为数据卡片/对比表格 |
